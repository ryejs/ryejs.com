Rye is a lightweight javascript library for DOM manipulation and events with support for all modern browsers including IE9+. It has support for touch events/gestures, an even emitter which you can use to extend your own objects, and a jQuery-compatible API.

Introduction

Rye's API is similar to jQuery but not totally compatible. Inconsistencies like .map() behavior, and map/each() argument ordering are fixed in Rye.

It doesn't do any gymnastics to subclass Array, instead elements are stored in the .elements property. This means that array methods on a Rye instance return another Rye instance, but you can access the elements array directly with .get() or .elements and use all the native Array methods (ex: $('li').each() vs $('li').get().forEach()).

Not going to happen

  • event aliases like delegate, live, click, focus
  • cancel bubbling and prevent default by returning false
  • effects and animation - do it with CSS
  • defining global $
  • traditional event namespacing
  • end() or other chain-aware methods

Traversal

find

.find(selector) ⇒ rye collection

Get the descendants of each element in collection, filtered by a string selector.

index

.index(selector) ⇒ number

Search for a given selector from among the collection.

If no argument is passed, the return value is the position number of the first element in collection relative to its sibling.

add

.add(elements) ⇒ rye collection
.add(Rye) ⇒ rye collection
.add(selector[, context]) ⇒ rye collection

Add elements to the set and returns a new collection.

pluckNode

.pluckNode(property) ⇒ rye collection

Using to create a new collection based in elements properties that returns a node. The method avoids return not elements and is called by .next() and .prev().

next

.next() ⇒ rye collection

Get the immediately following sibling of each element in collection.

prev

.prev() ⇒ rye collection

Get the immediately preceding sibling of each element in collection.

siblings

.siblings() ⇒ rye collection

Get the siblings of each element in collection.

parent

.parent([selector]) ⇒ rye collection

Get the parent of each element in collection, optionally filtered by a selector.

parents

.parents([selector]) ⇒ rye collection

Get the ancestors of each element in collection, optionally filtered by a selector.

closest

.closest(selector) ⇒ rye collection

For each element in collection, get the first element that matches the selector by testing the element itself and traversing up through its ancestors.

children

.children([selector]) ⇒ rye collection

Get the children of each element in collection, optionally filtered by a selector.

Collection

get

.get(index) ⇒ element
.get() ⇒ array

Returns the element that have the index in collection. If called without argument returns a copy of elements array.

eq

.eq(index) ⇒ rye collection
.eq() ⇒ empty rye collection

Performs .get() and wrap the element result in a rye collection. If called without argument returns empty rye collection.

forEach

.forEach(callback[, thisArg]) ⇒ array

Performs Array.prototype.forEach in elements of collection. The callback is invoked with: element value, element index and array.

reduce

.reduce(callback[, initialValue]) ⇒ array

Performs Array.prototype.reduce in elements of collection. The callback is invoked with: previous value returned, current element, element index and array.

reduceRight

.reduceRight(callback[, initialValue]) ⇒ array

Same as .reduce() except that performs in the right to left in array values.

indexOf

.indexOf(element[, fromIndex]) ⇒ array

Performs Array.prototype.indexOf in elements of collection. Returns the first index at which a given element can be found in elements, or -1 if it is not present.

map

.map(callback[, thisArg]) ⇒ rye collection

Creates a new rye collection with the results of calling callback on every element in collection. The callback is invoked with: element value, element index and array.

sort

.sort([fn]) ⇒ rye collection

Creates a sorted rye collection, optionally ordered by fn.

each

.each(callback) ⇒ self

Performs Array.prototype.forEach in elements of collection. The callback is invoked with: element value, element index and array.

iterate

.iterate(fn, context) ⇒ function

Returns a funcion that will call fn in given context passing each one element of collection as first parameter follow by arguments.

push

.push(element) ⇒ index

Add an element to collection and returns it index. This method changes the rye collection.

slice

.slice(start, end) ⇒ rye collection

Returns a new sliced rye collection.

concat

.concat(elements, ...) ⇒ rye collection

Concats array of elements or rye collections into a single rye collection.

pluck

.pluck(property) ⇒ array

It is just a wrap to util.pluck(elements, property) that iterate over collection.

put

.put(property, value) ⇒ array

It is just a wrap to util.put(elements, property) that iterate over collection.
